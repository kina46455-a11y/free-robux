# 🎮 Générateur de faux codes "Free Robux" (Éducatif)

Projet **HTML / CSS / JavaScript** qui simule un générateur de codes façon Roblox.  
⚠️ **Attention** : tous les codes générés sont **fictifs** et n’ont aucune valeur réelle.  
Ils sont uniquement là pour **apprendre à manipuler l’UI, le DOM, et GitHub Pages**.

---

## 🚀 Démo en ligne
👉 [Voir le site sur GitHub Pages](https://kina46455-a11y.github.io/free-robux/)  

---

## 📂 Structure du projet
- `index.html` → page principale  
- `style.css` → styles façon Roblox sombre avec accent Robux vert  
- `script.js` → génération aléatoire de faux codes + boutons (copier, télécharger, tester)  

---

## ⚙️ Fonctionnalités
- Générer des faux codes avec plusieurs formats (`RBLX-XXXX-XXXX-XXXX`, etc.)  
- Copier tous les codes en un clic  
- Télécharger les codes au format `.txt`  
- Simulation d’un champ *Redeem* (affiche un message explicatif)  
- UI inspirée de Roblox (mode sombre + vert Robux)  

---

## 📖 Usage pédagogique
Ce projet sert uniquement :  
- à apprendre **HTML / CSS / JavaScript**,  
- à découvrir **GitHub Pages**,  
- à expérimenter la génération et manipulation du DOM.  

**Il n’est pas affilié à Roblox et ne fournit aucun Robux réel.**
